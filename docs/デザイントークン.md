# てくしぇあ（TekuShare） デザイントークン・定数定義

> 色・サイズ・余白・フォント・アニメーションなど、アプリ全体で使う値をすべてここで管理する。
> **マジックナンバー禁止。数値を直書きしない。必ずここの定数を使う。**

## 使い方（NG例 → OK例）

```dart
// ❌ マジックナンバー直書き禁止
Container(
  padding: EdgeInsets.all(16),
  decoration: BoxDecoration(
    color: Color(0xFF2E7D32),
    borderRadius: BorderRadius.circular(12),
  ),
  child: Text('行きたい', style: TextStyle(fontSize: 15, color: Colors.white)),
)

// ✅ 定数を使う
Container(
  padding: const EdgeInsets.all(AppSpacing.lg),
  decoration: BoxDecoration(
    color: AppColors.primary,
    borderRadius: BorderRadius.circular(AppRadius.md),
  ),
  child: const Text('行きたい', style: AppTextStyle.bodyMedium),
)
```
