# てくしぇあ（TekuShare）

散歩中に「ここいいな」を1タップで記録できる Flutter アプリ。
ルートも自動で残るので、行きたい場所と歩いた道を地図で振り返れます。

## セットアップ

```bash
flutter pub get
flutter run --flavor dev -t lib/main_dev.dart
```

必要環境は [docs/環境構築手順.md](docs/環境構築手順.md) を参照。
